采用bindService 的方式，启动Service服务，与Activity 进行通讯，
通常在一些开发当中，需要采用Service 服务中的一些方法，或者是
取到Service 服务中的数据，在Activity 中进行界面的刷新！
为此，我们可以使用bindService 的方式，启动服务，以此达到想要的
目的，当然，根据此方法的实现，我们也可以使用 .aidl 文件进行数据
的传递，等等操作
在这里主要是做下简单的Demo示例！
